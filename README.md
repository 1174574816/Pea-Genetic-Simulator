# 🌱 豌豆遗传密码模拟器 · 孟德尔 3:1 定律

**一个交互式 Web 模拟器，通过随机杂交实验直观演示孟德尔遗传规律（高茎:矮茎 = 3:1），融合生物与数学教育。**

## 📖 项目背景

孟德尔通过豌豆杂交实验揭示了遗传因子的分离定律。本模拟器将这一经典实验数字化：

- **亲本**：杂合子高茎豌豆 (Aa) × 杂合子高茎豌豆 (Aa)
- **子代表现型**：高茎 (AA 或 Aa) 与矮茎 (aa)
- **理论比例**：高茎 : 矮茎 = 3 : 1（75% : 25%）

通过大量随机模拟，用户可以观察到实验比例逐渐收敛于理论值，理解大数定律与遗传概率的本质。

---

## ✨ 功能特性

- 🧪 **可调参数** – 设定模拟代数（10~500次）、模拟速度（慢/中/快/极速）
- 📊 **动态图表** – 实时绘制高茎/矮茎比例曲线，并与理论值（75%/25%）对比
- 📈 **统计看板** – 显示当前高/矮茎数量及比例
- 📝 **实验日志** – 记录关键代数的杂交结果，便于教学分析
- ⌨️ **键盘支持** – 空格键暂停/继续，`Ctrl+R` 重置实验
- 📱 **响应式布局** – 适配桌面与移动设备
